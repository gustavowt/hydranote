Features

Everything you need in one workspace

A complete toolkit for managing, understanding, and creating documents with AI assistance.

💬

Multi-Provider AI Chat

Connect to OpenAI, Anthropic, Google Gemini, Ollama, or run GGUF models locally. Switch providers freely.

🔍

Semantic Search

Vector embeddings index your documents for meaning-based retrieval across entire projects.

📄

Document Ingestion

Import PDF, DOCX, TXT, Markdown, and images with OCR. All content is chunked and searchable.

✏️

Markdown Editor

Full editor with edit, split, and preview modes. Mermaid diagrams, syntax highlighting, and export.

🛠️

AI Tool Execution

The LLM can read, search, summarize, write, update files, and research the web on your behalf.

📊

Version History

Track every file change with diff-based version storage. Restore any previous version.

🌐

Web Research

AI-powered web search via SearXNG, Brave, or DuckDuckGo. Results are cached and semantically indexed.

🔌

MCP Server

Expose your workspace to external LLM tools like Claude Desktop via a local MCP server.

Integrations

Works with your favorite AI

LLM and embedding providers are configured independently — mix and match as needed.
